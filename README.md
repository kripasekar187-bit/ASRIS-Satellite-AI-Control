# ASRIS: AI-Powered Satellite Risk Intelligence System 🛰️🌌

ASRIS is a professional-grade **Predictive Mission Control Dashboard** that uses Machine Learning to monitor space weather and protect satellites from orbital hazards.

### 🚀 Key Features:
- **Predictive Forecasting:** Forecasts telemetry trends for the next 10 seconds.
- **Explainable AI (XAI):** Shows the root cause of every risk alert.
- **Autonomous Mitigation:** Triggers emergency safety protocols and voice alerts.
- **NASA Mission UI:** A vibrant, high-tech dashboard for real-time monitoring.

### 🛠️ Tech Stack:
- **Language:** Python
- **AI Framework:** Scikit-Learn
- **Dashboard:** Streamlit
- **Visuals:** Plotly
